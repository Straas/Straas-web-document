Endpoint

GET /api/v1/app/monetization_rules

Parameters

Name Description

page Page of results to fetch
per_page Number of results to return per page
sort Data sorted with {+,-} by which column {monetizable_id}. Apply - to sort in descending order. For example, sort=-column
